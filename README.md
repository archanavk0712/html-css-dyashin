# HTML & CSS Practice – Dyashin Internship

This repository contains the HTML and CSS practice exercises completed during my training and internship preparation at Dyashin TechnoSoft Pvt Ltd.

The projects focus on building a strong foundation in web page structure, layout design, and responsive styling using HTML and CSS.

## Topics Covered

HTML Fundamentals

* HTML document structure
* Semantic HTML elements
* Forms and input elements
* Tables and lists
* Linking and navigation

CSS Fundamentals

* Selectors and properties
* Box model
* Layout techniques
* Styling text and components
* Flexbox and basic responsive design

## Technologies Used

HTML
CSS

Development Tools
Visual Studio Code
Browser Developer Tools

## Purpose of the Repository

This repository serves as practice for understanding how to structure and style web pages before moving into JavaScript and frontend frameworks.

## Learning Outcomes

* Understanding of HTML page structure
* Ability to design layouts using CSS
* Building responsive UI components
* Preparing for modern frontend frameworks such as React
